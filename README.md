# azure-blob-migration-pipeline

Project Overview

This project demonstrates a real-world DevOps scenario where a file from a simulated on-premise system is securely transferred to Azure Blob Storage using Azure CLI and authenticated cloud access methods.

It showcases foundational Cloud + DevOps skills including authentication, storage operations, and data migration workflows.

Problem Statement

The organization needed a reliable way to migrate files from an on-premise system to Azure Blob Storage for centralized cloud storage and future processing.

This project simulates that migration by uploading a local file (/tmp/nautilus.txt) into an Azure Blob container.

Tools & Technologies
Azure CLI
Azure Blob Storage
Azure Service Principal Authentication
Bash (automation script)
Linux environment
